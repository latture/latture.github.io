void coptim::Minimizer::operator() ( column_vector &  x)
inline

Minimizes an objective function.

The objective function is minimized starting from the coordinates specified in x using the specified search strategy. The search strategy is confined within the bounds of upper and lower: lower(i) < x(i) < upper(i)

Parameters
xcolumn_vector. Starting point for local minimization. x is modified in place.
