Detailed Description

Stores the output of an optimization.

Provides a consistent way of reporting the output of optimizations.

Parameters
xcolumn_vector. The coordinates of the best solution.
costdouble. The cost associated with the coordinates x.
messagestd::string. Message that describes how the optimization algorithm terminated.
successbool. Whether the solution had converged.
niterint. Number of iterations performed.
