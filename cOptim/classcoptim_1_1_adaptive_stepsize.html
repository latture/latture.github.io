void coptim::AdaptiveStepsize::_adjust_step_size ( )
inline

Adjusts the current step size based on the percentage of candidate solutions being accepted.

Step size is modified according to:

if (current_accept_rate > target_accept_rate) { step_size /= factor; }
else { step_size *= factor; }
if (step_size > max_step_size) {step_size = max_step_size;}

Generates a candidate solution.
